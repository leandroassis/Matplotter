                                                      ------------- AJUDA --------------
Bem vindo à ajuda do sistema. Nessa parte você encontrará a explicação de como utilizar o software, operadores, equações, método de funcionamento e os limites atuais do software.
   
                       				----------------  Operadores -----------------
As equações nesse programa seguem a regra de escrita em forma de código, porém só podem ser escritas em função de 'x' (ex: x**2+2*x ou x**2/(x-1)).

Os operadores suportados pelo software são:

+ Soma
- Subtração
/ Divisão
* Multiplicação
** Exponenciação
exp(n) e elevado a n
sin(x) função seno de x
cos(x) função cosseno de x
tg(x) função tangente de x
sinh(x);cosh(x);tgh(x) - funções hiperbólicas de x
sqrt() raiz quadrada
root(x, n) raiz n-esíma de x

Não são aceitas operações de divisão inteira (//) ou obtenção de resto (%). Os operadores inseridos por parte do usuário não são utilizados como forma de ordenar a realização de um cálculo, e sim formar a equação que o programa deve usar como base para realizar a tarefa. O software consegue processar algumas funções compostas.

   						----------------- Funções -------------------
Este programa trabalha bem com a maior parte das funções trigonométricas e polinomiais, incluindo funções com descontinuidade infinita. Porém não é possível realizar algumas operações com funções compostas; funções trigonométricas com alto grau de complexidade; funções exponenciais de base nepteriana e expoentes com alto grau de complexidade; funções polinomiais descontínuas em mais de 3 pontos; funções com números complexos ou que resultem nestes (cálculo de volumes/áreas); funções com mais de uma variável; dentre outros casos não mapeados.

OBS: Podem ocorrer mudanças de perfomance dependendo da tarefa e da função inserida. É comum um tempo de processamento de até 15 segundos para gerar o gráfico de funções trigonométricas, complexas ou não.

         ----------------- Possíveis Problemas --------------
Não conseguir realizar o esboço do gráfico - verifique a equação, parenteses e a ordem das operações.
Gráfico bagunçado ou reta (ruídos) - Erro e falhas causadas por conta da limitação do programa, basta tentar outra função.
Erro repetidamente ao realizar ajuste com dados manuais - verifique se está inserindo a mesma quantidade de x e y
Gráfico do ajuste com enquadramento defeituoso - Erro de código, apenas tente outros dados ou método de ajuste.
Erro repetidamente ao usar dados de planilhas - Verifique os nomes das colunas que deseja usar; modifique o nome da coluna para que não tenha acentos ou caracteres especiais; verifique o nome e caminho até o
arquivo.
